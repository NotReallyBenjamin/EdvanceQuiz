Listening Self-Assessment Website

Files included:
- index.html: the website

How to use:
1. Upload index.html to a simple website host such as Netlify, GitHub Pages, Google Sites, or your organisation's web server.
2. Copy the public website link.
3. Create a QR code from that link using any QR code generator.
4. Learners scan the QR code and complete the assessment on their phone.

Note:
A QR code needs a public web link. This file will work immediately once hosted.
